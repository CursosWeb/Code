## De texto a Habla

Uso de módulos de texto a habla: [say.py](texto-habla/say.py)

Dependencias:

* [pyttsx3](https://pypi.org/project/pyttsx3/)

```commandline
pip install pyttsx3
```

Otros módulos que se pueden usar son:

* [gTTS](https://pypi.org/project/gTTS/). Usa la API de Google Translate.
