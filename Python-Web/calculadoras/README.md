# Calculadoras

Soluciones parciales al ejercicio "Calculadoras":

* `spec.md`: Especificación, utilizada para producir `calculadoras.py` con un prompt similar a "Genera el programa especificado en spec.md".
* `calculadoras.py`: Programa generado a partir de la especificación
* `test_calculadoras.py`: Programa de pruebas (tests) generado a partir de la especificación y del programa generado, con un prompt similar a "Genera tests para calculadoras.py, teniendo en cuenta la especificación spec.md, suponiendo que el programa ya ha sido lanzado".
