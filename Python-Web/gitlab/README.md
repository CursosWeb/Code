# Interfaz para GitLab

Soluciones parciales al ejercicio "Sitio web para gestionar issues de GitLab"

A continuación, ejemplos de ejecuciones de los programas. Para usar token
de acceso a GitLab, crear un fichero `.env` con el token en el directorio
(ver `.env.example`).

```bash
python issues.py
```

```bash
python gitlab.py list
```

```bash
python gitlab.py create "Título del issue"
```

```bash
python translate.py Hola qué tal
```

```bash
python server.py
```

```bash
python server2.py
```
